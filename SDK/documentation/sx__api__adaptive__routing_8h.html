This API Initiates the adaptive routing (AR) module in SDK. This function should be called before any use of the library.

Supported devices: Spectrum2, Spectrum3.

[in]init_params_p- General AR parameters.
Returns

SX_STATUS_INVALID_HANDLE if a NULL handle is received.
SX_STATUS_PARAM_NULL if parameter is NULL.
SX_STATUS_PARAM_ERROR if parameter is wrong.
SX_STATUS_ALREADY_INITIALIZED - When AR is already initialized.
SX_STATUS_MODULE_UNINITIALIZED – if router module isn’t initialized.
SX_STATUS_ERROR general error.

◆ sx_api_ar_congestion_threshold_set()

sx_status_t sx_api_ar_congestion_threshold_set ( const sx_api_handle_t  handle,
const sx_access_cmd_t  cmd,
const sx_ar_congestion_threshold_attr_t *  threshold_p 
)

This API sets the AR congestion threshold configuration. This configuration will impact the port grade. The grades are used by the AR engine to select the egress port.

Buffer < thresh_lo will get grade 0/1 depends on link utilization thresh_lo < Buffer < thresh_med will get grade 2 thresh_med < Buffer < thresh_hi will get grade 3 thresh_hi < Buffer will get grade 4

Note: min threshold = 0, max threshold = 2^24

Supported devices: Spectrum2, Spectrum3.

Parameters
[in]handle- SX-API handle
[in]cmd- SET.
[in]threshold_p- Congestion threshold.
Returns
SX_STATUS_SUCCESS if operation completes successfully
SX_STATUS_CMD_UNSUPPORTED if access command isn't supported.
SX_STATUS_INVALID_HANDLE if a NULL handle is received.
SX_STATUS_ERROR for a general error
SX_STATUS_PARAM_ERROR if any input parameter is invalid.
SX_STATUS_PARAM_NULL if any input/output parameter is NULL.
