typedef struct sx_truncation_profile_parsing_labels_capabilities sx_truncation_profile_parsing_labels_capabilities_t

This struct store Masking Labels which can be configured for Parsing Base masking.

The packet will be masked from the beginning of the matching label. For example, if L3 was selected, packet will be masked post L2 header. When a packet has a match to multiple labels enabled, the header with max offset will be selected.
