541 /***
542  * This API adds/edits/deletes a Layer 4 port range comparison set (up to SX_ACL_MAX_PORT_RANGES).
543  *
544  * ADD command is used to create a new range ID from the supplied port range.
545  * EDIT command is used to update the port range of the supplied range ID.
546  * DELETE command is used to clear the range configuration. This will fail if the range is being used by an ACL.
547  *
548  * Note: SX_ACL_PORT_RANGE_IP_HEADER_BOTH option for IP header type is not supported in Spectrum2 and Spectrum3 systems.
549  *
550  * Supported devices: Spectrum, Spectrum2, Spectrum3.
551  *
552  * @param[in] handle - SX-API handle
553  * @param[in] cmd - ADD/EDIT/DELETE
554  * @param[in] l4_port_range - Layer 4 port range comparison
555  * @param[in, out] range_id - Port range comparison ID
556  *
557  * @return SX_STATUS_SUCCESS Operation completed successfully
558  * @return SX_STATUS_MESSAGE_SIZE_ZERO Message size error
559  * @return SX_STATUS_MESSAGE_SIZE_EXCEEDS_LIMIT Message size error
560  * @return SX_STATUS_PARAM_NULL, SX_STATUS_PARAM_ERROR or SX_STATUS_PARAM_EXCEEDS_RANGE Any input parameter is invalid
561  * @return SX_STATUS_ENTRY_NOT_FOUND ACL element is not found in database
562  * @return SX_STATUS_SXD_RETURNED_NON_ZERO Hardware failure
563  * @return SX_STATUS_CMD_UNSUPPORTED Command is unsupported
564  * @return SX_STATUS_INVALID_HANDLE Invalid handle error
565  */
