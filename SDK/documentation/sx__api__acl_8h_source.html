525 /***
526  * This API adds/edits/deletes a Layer 4 port range comparison set (up to SX_ACL_MAX_PORT_RANGES).
527  *
528  * ADD command is used to create a new range ID from the supplied port range.
529  * EDIT command is used to update the port range of the supplied range ID.
530  * DELETE command is used to clear the range configuration. This will fail if the range is being used by an ACL.
531  *
532  * Note: SX_ACL_PORT_RANGE_IP_HEADER_BOTH option for IP header type is not supported in Spectrum2 and Spectrum3 systems.
533  *
534  * Supported devices: Spectrum, Spectrum2, Spectrum3.
535  *
536  * @param[in] handle - SX-API handle
537  * @param[in] cmd - ADD/EDIT/DELETE
538  * @param[in] l4_port_range - Layer 4 port range comparison
539  * @param[in, out] range_id - Port range comparison ID
540  *
541  * @return SX_STATUS_SUCCESS Operation completed successfully
542  * @return SX_STATUS_MESSAGE_SIZE_ZERO Message size error
543  * @return SX_STATUS_MESSAGE_SIZE_EXCEEDS_LIMIT Message size error
544  * @return SX_STATUS_PARAM_NULL, SX_STATUS_PARAM_ERROR or SX_STATUS_PARAM_EXCEEDS_RANGE Any input parameter is invalid
545  * @return SX_STATUS_ENTRY_NOT_FOUND ACL element is not found in database
546  * @return SX_STATUS_SXD_RETURNED_NON_ZERO Hardware failure
547  * @return SX_STATUS_CMD_UNSUPPORTED Command is unsupported
548  * @return SX_STATUS_INVALID_HANDLE Invalid handle error
549  */
